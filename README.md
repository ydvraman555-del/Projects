# Power BI Data Analysis & Dashboard Project

## Overview
This project focuses on data analysis and interactive dashboard creation using **Power BI Desktop**.
It aims to analyze the dataset, identify trends, and present insights through clear and dynamic visualizations.

## Project Files
- Power BI report file (.pbix)
- README.md

## Dataset
- Source: Public / Kaggle dataset
- Type: Structured dataset
- Purpose: Analytical reporting and visualization

## Tools & Technologies
- Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)

## Work Performed
- Data cleaning and transformation
- Data modeling and relationships
- Creation of calculated columns and measures
- KPI analysis and reporting
- Interactive dashboards and visuals

## Dashboard Features
- Slicers and filters
- Bar charts, line charts, cards, and tables
- KPI indicators
- Drill-down and cross-filtering

## How to Use
1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Use filters and visuals to explore insights

## Future Enhancements
- Advanced DAX measures
- Additional dataset integration
- Publish to Power BI Service
- Scheduled data refresh
